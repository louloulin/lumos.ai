# LumosAI-UI 开发文档

## 📋 概述

LumosAI-UI 是一个现代化的AI应用界面系统，基于Dioxus框架构建，支持Web和Desktop双平台。本文档记录了项目的开发历程和技术实现。

> **注意**: 本文档已重构，详细的功能状态请查看 [ui_status.md](ui_status.md)

## 🚀 启动机制

### 支持的启动模式
```bash
# Web 模式 (默认)
cargo run --bin lumosai-web-server

# Desktop 模式
cargo run --bin lumosai-desktop --features desktop

# Fullstack 模式 (SSR)
cargo run --bin lumosai-web-server --features fullstack
```

### 技术架构
```
lumosai_ui/
├── web-pages/          # UI 组件层 (Dioxus)
├── web-assets/         # 前端资源层
├── web-server/         # 后端服务层 (Axum)
└── examples/           # 示例代码
```
